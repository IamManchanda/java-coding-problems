# Check Permutation

> Question from Cracking the Coding Interview, Chapter 1.

Given two strings, write a method to decide if one is a permutation of the other.

Example:

```plaintext
Input: "abc", "bac"
Output: true
```

```plaintext
Input: "123", "321"
Output: true
```

```plaintext
Input: "harry", "larry"
Output: false
```
